Place talking-head MP4s here to upgrade the avatar from 2D SVG to a real
3D video avatar:

  maya.mp4   -> female teacher (default)
  leo.mp4    -> male teacher

Sources that work well:
  - D-ID (did.com) — generate from a portrait photo
  - HeyGen (heygen.com) — prebuilt avatars
  - Synthesia (synthesia.io)
  - Any looping 3D render exported to MP4

Recommended specs:
  - 512x512 or square aspect, <= 6 MB
  - Looping talking motion with neutral expression
  - No audio (the mock AI uses text; hook up TTS later if desired)

Without these files, the app falls back to an animated SVG avatar that
still moves its mouth while the teacher "speaks".
